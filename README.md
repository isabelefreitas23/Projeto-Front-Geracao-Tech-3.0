# Digital Store - Front-end

Projeto front-end de e-commerce desenvolvido com React como projeto final do curso Fullstack da Geração Tech 3.0.

## Tecnologias utilizadas
- React
- Vite
- React Router
- Tailwind CSS

## Como rodar o projeto

Clone o repositório:

git clone https://github.com/isabelefreitas23/Projeto-Front-Geracao-Tech-3.0.git

Instale as dependências:

npm install

Execute o projeto:

npm run dev
